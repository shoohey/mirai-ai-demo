3D Avatar (VRoid Studio + Unity)

現在のPNGアバターを3Dアニメーションに進化させましょう！
VRoid Studioでキャラクターを作り、Unity WebGLでビルドするだけです。

  1. VRoid Studioをダウンロード（無料）
    vroid.com/studio
  2. メイドカフェキャラクターを作成
    髪型、目、服装をカスタマイズ。メイド服のプリセットも利用可能。
    表情を設定（笑顔、驚き、考え中など）
  3. VRMファイルとしてエクスポート
    ファイル → VRMエクスポート → ポリゴン数を「中」に設定
  4. Unity 2022.3 LTSでプロジェクトを開く
    unity-project/ フォルダにUnityプロジェクトが用意済み
  5. VRMファイルをUnityにインポート
    UniVRM パッケージをインストール → VRMファイルをドラッグ&ドロップ
  6. WebGLビルド
    File → Build Settings → WebGL → Build
    出力先: public/unity/Build/
  7. 自動切替！
    ビルドファイルを配置するだけで、システムが自動的に3Dアバターに切り替わります。

3Dアバターのメリット：

  • リアルタイムリップシンク（音声に合わせて口が動く）
  • 自然なまばたき・呼吸アニメーション
  • 表情ブレンドシェイプ（喜び・驚き・考え中の滑らかな切替）
  • お客さんへのインパクト大幅UP
